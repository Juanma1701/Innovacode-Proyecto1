# Innovacode ERP

Aplicación móvil para la gestión de inventarios, ventas y operaciones en pequeños negocios como minimercados, licorerías y zapaterías.

## 🚀 Funcionalidades
- Gestión de inventario
- Registro de ventas
- Control de compras
- Seguimiento de pedidos
- Autenticación de usuarios

## 🛠 Tecnologías
- Kotlin
- Android Studio
- Firebase (en proceso)

## 📦 Módulos
- Auth (login/registro)
- Dashboard
- Inventario
- Ventas
- Compras
